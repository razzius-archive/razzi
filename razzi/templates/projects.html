Ideas


Recyclepedia

Recyclepedia would be a Penn-specific web resource for determining how to properly dispose of anything. The main functionality would be an index for looking up items, a map of recycling bins / ewaste / compost locations, a FAQ about recycling on campus.

Recursive Learning

Have you ever done a tutorial that either skipped setup information that you needed as a beginner? How about one where they spent too much time on things like project setup, when you wanted to jump right into the implementation?

The idea of Recursive Learning would be to allow beginners to expand any instruction, and experienced users to collapse them. An experienced user doing a Node tutorial, for example, could hide all of the Node installation steps, and a beginner to iOS development could expand a section like "Create a project using the XCode template" to any amount of detail--instructions like "Installing XCode" and "Using XCode project templates" would first appear, with those expandable, recursing to single actions like clicking on dialog boxes.

Campus Dash

This simple application would organize online resources of a university, such as their BlackBoard site, their course registration website, and student news on a dashboard that would emphasize usability. Many colleges could have their own subdomain. The goal would be to have students set this dashboard as their homepage and get traffic whenever they open their browser.


Any of this sound interesting? Check out my Github to check the status of projects like these, and email me at razzi53@gmail.com if you'd like to collaborate, or if you'd like to share your own ideas.
